# KISI_Project
사내 일정관리 앱 - KISI는 지인의 회사에서 사용하고 있는 Android Native Application 입니다.   
Firebase Realtimebase에 사내 공지사항 및 일정을 CRUD 하는 것이 주된 기능입니다.   
`Kotlin`, `Firebase`

## Environments
- Windows 10 (Desktop, Laptop)
- Android Studio
- Kotlin
- XML
